# Docs

`docs/` 是 `Memknow` 的文档入口目录。

如果你通过 GitHub Pages 浏览本项目，默认入口是 [index.html](./index.html)；如果你是在仓库里直接看文件，建议先从下面几个文档开始：

- [course/](./course/)
  面向进阶用户的系统导览页，按主链路、workspace、session、长期记忆、heartbeat 和 schedule 解释项目如何运作。
- [README.md](https://github.com/ashwinyue/Memknow/blob/main/README.md)
  项目总览、能力列表、架构概览和基础使用方式。
- [design.md](https://github.com/ashwinyue/Memknow/blob/main/docs/design.md)
  更贴近实现细节的系统设计说明，适合对照代码阅读。
- [quickstart.md](https://github.com/ashwinyue/Memknow/blob/main/docs/quickstart.md)
  部署与运行步骤。

其他文档：

- [configuration.md](https://github.com/ashwinyue/Memknow/blob/main/docs/configuration.md)

- [windows-compatibility.md](https://github.com/ashwinyue/Memknow/blob/main/docs/windows-compatibility.md)

1. `course/`
